EstimateR {EpiEstim}R Documentation

Function to ensure compatibility with EpiEstim versions <2.0

Description

Please only use for compatibility; Prefer the new estimate_R function instead

  Mean.SI = NULL, Std.SI = NULL, Std.Mean.SI = NULL,

  Mean.Prior = 5, Std.Prior = 5, CV.Posterior = 0.3, plot = FALSE,
  leg.pos = "topright")
